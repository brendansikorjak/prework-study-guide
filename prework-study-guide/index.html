HTML

  • The head element contains information about the webpage.
  • title is a rare visual head element that appears in the browser title bar.
  • SEO (Search Engine Optimization) helps rank your website in search engines.
  • title element, alt attribute both help improve search engine rankings
  • The body element represents the visible content shown to the user.
  • h1, the largest heading size, - h6, the smallest heading size
  • p element creates a paragraph or block of text
  • ul element is an unordered list, ol ordered list, and li list item
  • img element contains info about images displayed
  • src attribute locates the image used
  • br element creates a line break
  • link element creates a connection to an external file
  • rel attribute specifies the relationship between current doc and linked doc resource
  • href attribute specifies the location of the external resource using a URL
  • Parent elements hold other elements within them
  • Child elements are the elements inside of parent elements

CSS

  • margin property indicates space around the outside of an element
  • margin: first value top and bottom second value left and right
  • padding property indicates space around the content inside an element
  • * selector applys a rule to all visible elements
  • You can apply a rule with mulitple selectors by seperating them with comma
  • Keep your code DRY aka Don't Repeat Yourself
  • text-align is a positioning property
  • Use px for sizing
  • Always make sure your css file is linked with in your html file using link element
  • A period preceding a selector means its is a class selector
  • Class selector rules only apply to html elements assigned to that class
  • box-shadow: x offsets to the right y offsets below the box third value is color
  • hex # is a specific color by hexadecimal code ex. #888888, #9370DB

  • git checkout -b branch-name: creates a new branch and switches to it
